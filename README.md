A Big-Beautiful-Ray-Tracer: I'm making this because I'm bored and it seems very fun to make.

# How to build & run

## First build
cmake -S . -B build
cmake --build build && ./build/ppm_gradient > image.ppm


## Re-Builds
cmake --build build && ./build/ppm_gradient > image.ppm

## Running
./my_program
